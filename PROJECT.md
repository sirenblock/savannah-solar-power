# Project: SavannahSolarPower.com - Complete Website Rebuild

## Company Overview
**Savannah Solar Power (C SQUARED POWER LLC)** is a residential + commercial solar installation company in Savannah, GA providing turnkey solar solutions — permitting, engineering, installation, interconnection, and post-installation service.

## Project Goal
Completely rebuild SavannahSolarPower.com into a detailed, SEO-optimized, high-converting website that clearly explains every service, product, and solution we actually offer. Use real company workflow, offers, invoicing history, equipment lines, and installation experience.

## Equipment We Actually Install

### Solar Panels
- **Jinko 425W** - High-efficiency monocrystalline, 25-year warranty
- **Longi LR5 400W** - Premium tier-1 manufacturer, excellent low-light performance
- **QCells 415W** - German-engineered, superior temperature coefficient

### Inverters
- **Enphase IQ8M** (default) - Microinverter, 384W max AC output, grid-forming technology
- **SolArk 12K / 15K / 20K** - Hybrid inverters, battery-ready, generator integration
- **EG4 hybrid inverters** - Cost-effective hybrid solutions, versatile configurations
- **Tesla Powerwall 3 inverter** - Integrated 11.5 kW continuous power
- **Franklin aGateX + aPower2** - Advanced whole-home backup, modular design

### Batteries
- **Tesla Powerwall 3** - 13.5 kWh, 11.5 kW continuous, all-in-one design
- **EP Cube 9.9 / 16.5 / 19.9 kWh** - Modular battery systems, scalable storage
- **EG4 PowerPro** - LiFePO4 chemistry, wall-mount or floor-mount
- **EG4 FlexBOSS** - Expandable battery system, long cycle life
- **Franklin aPower2** - 13.6 kWh per unit, stackable up to 4 units

### Racking Systems
- **IronRidge XR100, XR10, FlashFoot2** - Industry-standard racking, 25-year warranty
- **IronRidge ground mount** - Ballasted or ground-screw options, engineered for wind loads
