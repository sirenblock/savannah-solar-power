### Desktop (1200px+)
- Full grid layouts (3-4 columns)
- Large hero text
- Spacious padding

### Tablet (768px - 1199px)
- 2-column grids
- Medium hero text
- Standard padding

### Mobile (< 768px)
- Single column stacking
- Smaller hero text
- Compact padding
- Full-width buttons
